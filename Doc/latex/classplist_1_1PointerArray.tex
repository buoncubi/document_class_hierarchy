\subsection{Detailed Description}
\subsubsection*{template$<$class T$>$class plist\-::\-Pointer\-Array$<$ T $>$}

This class describes a manager of a dynamic array of generic type T$\ast$. It manage manipulation, creation, copy and destruction, as well as basic operator overloading. In dettails, the array is specified by a pointer to the head, a size integer and a cnt meaning the number of elements currently in the array.
